# BookVault

BookVault — веб-приложение для управления личной книжной коллекцией. Проект позволяет пользователю создавать каталог книг, разделять записи на публичные и приватные, просматривать доступные книги сообщества и работать с базовой моделью читателей.

Приложение решает задачу простой персональной библиотеки с авторизацией, серверными API и хранением данных в Supabase/PostgreSQL.

## Основные функции

- Авторизация пользователей через Supabase Auth: регистрация, вход, выход и хранение пользовательской сессии.
- Управление книгами: создание, просмотр, удаление и API-обновление записей.
- Разделение книг на публичные и приватные с фильтрацией по доступности.
- Просмотр каталога книг с поиском по названию/описанию и постраничной загрузкой.
- Базовая модель читателей: таблицы `readers` и `book_readers`, отображение связанных читателей на странице книги.

## Технологический стек

- **Язык:** TypeScript
- **Frontend:** Next.js 14, React 18, App Router
- **UI:** Tailwind CSS, локальные UI-компоненты, Lucide React, clsx
- **Backend:** Next.js API Routes
- **База данных и Auth:** Supabase, PostgreSQL, Row Level Security
- **Валидация:** Zod
- **Инструменты разработки:** ESLint, PostCSS, Autoprefixer
